O trabalho foi desenvolvido em ambiente linux, por isso todos os comandos que se 
seguem poderão não funcionar noutros sistemas operativos. É necessário o python
2.7. No sistema operativo usado o mesmo já vinha instalado por defeito, por esse
motivo não é apresentado um modo de instalar o mesmo. 

INSTALAÇÃO DE PACKAGES NECESSÁRIOS À EXECUÇÃO DO PROGRAMA:
- sudo apt-get install python-pip
- sudo pip install boto
- sudo pip install topia.termextract
- sudo pip install tweepy

EXECUÇÃO DO PROGRAMA:
- Depois de extrair o ficheiro .zip com os ficheiros, entre na pasta onde se
encontram os ficheiros de código .py.
- Execute o terminal nesse diretório
- Escreva o seguinte comando para correr a aplicação: python Client.py
